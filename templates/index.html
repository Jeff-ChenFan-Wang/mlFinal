About Live Demo

Live BMI

Welcome to Jeff's Live BMI demo!

This project is meant to replicate the results of Face-to-BMI: Using Computer Vision to Infer Body Mass Index on Social Media (Kocabey et al. 2017).

 

Web App Architecture

Frontend webcam streaming is done locally on the client side via javascript. This makes the app much more responsive and self contained, instead of needing to mess with STUN servers and having WebRTC bog down our backend. Backend is supported by Flask and Gunicorn deployed on a GCP Compute Engine instance.

 

Backend Model

The model uses Haar Frontal Face Cascade Classifier to recognize multiple faces in the webcam. The recognized faces are then extracted for BMI prediction.

A Resnet-50 model trained on the VGG-Face (Parkhi et al. 2015) dataset for facial recognition is used to extract face embeddings from the faces recognized. The embeddings are then used for BMI prediction. The original paper then fed the embeddings through an epsilon support vector regression model to predict subject BMI. Disclaimer

This is mostly for fun to test model building and deployment capabilities. The author of the paper themselves say that any machine learning prediction method of individual characteristics may work “better than random guessing”, but would still only be useful for studying the general population and would be wildly inaccurate for any individual.
